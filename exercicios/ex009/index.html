Formatações

Hoje vamos nos aprofundar em algumas formatações importantes e essas são as minhas anotações de estudo:

Negrito / Destaque:

Nesta frase temos um termo em negrito utilizando a tag < b > a tag é não semantica,ou seja ela não tem significado, ela só tem forma

Nesta frase tambem temos um termo em destaque só que agora utilizando a tag strong e ela é semantica
Visulmente a não temos diferença só que a formatação com b (Mencionada acima), ela so tem formatação de forma . O strong além da forma, ele dá um sentido

Itálico:

Nesta frase, temos um termo em itálico o usando a tag i (não semantica)

Nesta frase, Também temos uma termo em enfase utilizando a tag EM

Texto Marcado:

Nesta frase temos um termo marcado usando a tag < MARK >

Texto grande e Texto pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo utilizando as tag < big> que é obsoleta e a tag < samll > (Não é obsoleta)

Texto deletado:

Podemos marcar um texto como excluido para indicar que deve ser lido, mas não considerado.
Aqui utilizamos a tag < del >

Texto Inseridos:

Podemos marcarr um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
Aqui utilizamos a tag < ins >

Existe também o sublinhado com a tag U (Não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3 utilizando a tag < sup >

Texto Subescrito

Para inserir coisas do tipo H2O, utulizando a tag < sub >
