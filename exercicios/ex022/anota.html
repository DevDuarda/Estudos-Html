🎨 Estilos Inline e Alinhamento no CSS

📝 O que são Estilos Inline?

Estilos inline são regras CSS aplicadas diretamente dentro da tag HTML usando o atributo style.

✔️ Úteis para pequenas edições rápidas.
❌ Não são recomendados para projetos grandes, pois dificultam a manutenção.
❌ Poluem o código e podem causar problemas de organização.
❌ Misturam conteúdo (HTML) com estilo (CSS), o que não é uma boa prática.
❌ Só devem ser usados em casos muito específicos.

💻 Como aplicar um estilo inline?

Adicionamos o atributo style dentro da tag HTML.

👉 Exemplo:

<p style="color: #f1aa31; font-family: Consolas;">Seu texto aqui</p>

📐 Alinhamento de Texto (text-align)

🔸 text-align: left; ➡️ Alinhado à esquerda (padrão).

🔸 text-align: right; ➡️ Alinhado à direita.

🔸 text-align: center; ➡️ Centralizado.

🔸 text-align: justify; ➡️ Justificado, alinhado dos dois lados, ideal para blocos de texto maiores.

🔧 Outras Propriedades CSS Inline

  • color: Cor do texto. (Ex.: color: #f1aa31;)
  • background-color: Cor de fundo.
  • font-family: Fonte do texto.
  • font-size: Tamanho da fonte.
  • text-align: Alinhamento (left, right, center, justify).
  • padding: Espaçamento interno.
  • margin: Espaçamento externo.
  • border: Cria uma borda ao redor do elemento.
